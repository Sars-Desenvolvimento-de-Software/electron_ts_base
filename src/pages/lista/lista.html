Bibliotecas Utilizadas

Stack principal e dependências que compõem o template Electron + Vite + ESM + TS

Electron

Framework para construção de aplicações desktop multi-plataforma usando tecnologias web (HTML, CSS e JS).

Vite

Ferramenta de build moderna, focada em velocidade e HMR instantâneo para o renderer.

TypeScript

Superset tipado de JavaScript utilizado em todo o projeto (main, preload e renderer).

vite-plugin-electron

Plugin que compila o processo principal e o preload usando esbuild, integrando-os ao fluxo do Vite.

vite-plugin-electron-renderer

Permite o uso direto de APIs do Electron no renderer quando necessário, sem bundling adicional.

wait-on

Utilitário que aguarda portas, URLs ou arquivos — usado para iniciar o Electron somente após o Vite subir.

npm-run-all

Executa múltiplos scripts em paralelo ou sequência — utilizado para orquestrar o Dev e Build.
