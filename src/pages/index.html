Visão geral

Projeto de exemplo com Electron (Main + Preload), renderer gerenciado pelo Vite e código em TypeScript — tudo usando ESM. Ideal para começar apps desktop modernos com rápido ciclo de dev.

Design & filosofia

Leveza, separação de responsabilidades e ferramentas modernas: Vite para a experiência do desenvolvedor, esbuild para builds rápidos do main/preload e TypeScript para robustez.

Comandos

# dev
npm run dev

# build
npm run build

# start (app empacotado)
npm run start

Dicas rápidas

  • Use wait-on para garantir que o Vite esteja pronto antes do Electron.
  • Coloque tipos para canais IPC para segurança ponta-a-ponta.
  • Separe configs: vite.config.ts (renderer) e electron.vite.config.ts (main/preload).
